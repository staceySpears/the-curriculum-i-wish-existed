# Track 0 Exercises

These exercises are small on purpose.

Track 0 is about proving that you can use the basic environment without relying on hidden assumptions.

## Exercise 1 — Folder map

Create a folder structure with at least two nested folders and two files.

Write a short map of where each file lives.

Evidence:

- folder exists
- files exist
- written path explanations are correct

## Exercise 2 — Terminal navigation

Use the terminal to move through the folder structure.

Evidence:

- list the commands you used
- explain what each command did
- identify your current directory at least twice

## Exercise 3 — Edit and verify

Open a file in your editor, change it, save it, and verify the change from the terminal or by reopening the file.

Evidence:

- changed file
- short explanation of how you verified the change

## Exercise 4 — Run something

Run the tiny program or starter script provided for Track 0.

Evidence:

- command used
- expected output
- actual output
- note about any error encountered

## Exercise 5 — Git snapshot

Create at least two commits.

Evidence:

- meaningful commit messages
- clean status after final commit
- short explanation of what Git tracked

## Exercise 6 — Debugging note

Create or encounter a small safe error and document how you resolved it.

Evidence:

- expected behavior
- actual behavior
- hypothesis
- fix
- what you would check first next time
