# Track 1 — Java and Data Structures

Track 1 is where the learner begins writing code intentionally.

This track introduces Java as a first serious language, data structures as a design tool, and algorithmic reasoning as a skill. The goal is not to memorize syntax. The goal is to build the habit of choosing the right tool for the right problem and being able to explain why.

## What this track teaches

- Java syntax and program structure
- How to read compiler and runtime errors
- Arrays, lists, maps, sets, stacks, and queues
- When to use each data structure and why
- Basic algorithmic complexity: O(n), O(log n), O(n²)
- Recursive thinking
- Writing simple unit tests
- Debugging Java programs
- Self-evaluation against a rubric

## AI policy

AI is not allowed for Track 1 curriculum work.

Do not use AI to generate code, explain your specific solution, write tests for you, debug your assignment, or complete the project.

Use official documentation, curriculum materials, human-written resources, and the community.

The reason is substrate. Track 1 builds the reasoning habits that make later AI use safe. If AI solves every problem here, you skip the most important part.

See `../../AI_POLICY.md`.

## Modules

Complete the modules in order. Modules cover Java basics, each major data structure, complexity analysis, recursion, and debugging patterns.

## Project

The Track 1 project is a small data-driven application or problem set that requires choosing and defending data structure decisions.

## Gate

The Track 1 gate proves that you can write working code, choose data structures intentionally, and explain the tradeoffs.

See `gate.md`.

## Track 1 definition of done

Track 1 is complete when you can:

- write and run Java programs
- use arrays, lists, maps, sets, stacks, and queues
- explain why you chose a data structure for a given problem
- describe the complexity of your solution
- write simple unit tests
- debug Java errors using the error message and stack trace
- evaluate your own code against a rubric
- pass the Track 1 gate
