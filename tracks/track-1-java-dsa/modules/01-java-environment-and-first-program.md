# Module 1 — Java Environment and First Program

## Goal

Set up a working Java development environment and run your first Java program.

## Prerequisites

- Track 0 complete

## What this module covers

- Installing the JDK and verifying the installation
- Understanding the relationship between `.java` source files and `.class` bytecode
- Writing and running a minimal Java program (`Hello, World`)
- Reading your first compiler error
- Understanding `public static void main(String[] args)` line by line
- The Java compilation and execution cycle: `javac` then `java`

## Spine prompts

### Discipline of Doubt

What assumptions are you making about which version of Java is installed? How do you verify them?

### Debugging

If `javac Hello.java` fails, what is the first thing you check? What does the error message tell you?

### Technical communication

Explain the difference between `javac` and `java` to someone who has never used a compiled language.
